# Pitang Commerce

## Descrição do projeto

O projeto consiste em uma aplicação web apenas para fins de estudo e aprofundamento em React js, onde consiste de uma landing page inicial e uma listagem de produtos.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas:

- **[React](https://reactjs.org/)** 
- **[Tailwind CSS](https://tailwindcss.com/)** 
- **[TypeScript](https://www.typescriptlang.org/)** 
- **[TanStack Router](https://tanstack.com/router)** 
- **[Shadcn](https://ui.shadcn.com/)**

## Como Executar 

1. Clone o repositório:
   ```bash
   git clone https://github.com/Gabrielgsw/pitang-class-2026
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd pitang-class-2026/packages/pitang-frontend
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Execute o servidor de desenvolvimento local:
   ```bash
   npm run dev
   ```

5. Acesse a aplicação no navegador em:
   ```
   http://localhost:5173
   ```

6. Para gerar o build de produção:
   ```bash
   npm run build
   ```
